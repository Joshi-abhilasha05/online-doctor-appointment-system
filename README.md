## Online Doctor Appointment System
A simple web-based doctor appointment system developed using HTML, CSS, JavaScript, and JSON. This project allows patients to book appointments with doctors through an easy-to-use interface without using any backend technologies.

### Features:
* Patient login and appointment booking
* Doctor login and appointment viewing
* Admin dashboard for managing users and appointments
* JSON-based local data handling

### Technologies Used:
* HTML
* CSS
* JavaScript
* JSON

### Objective:
To understand frontend development, DOM manipulation, and JSON-based data handling by simulating a real-world healthcare appointment system.
